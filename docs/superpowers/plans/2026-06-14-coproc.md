> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Implement the `coproc` coprocess — `coproc command` (anonymous `COPROC`) and `coproc NAME compound` (named) — spawning an async command with two pipes wired to a `NAME[0]`/`NAME[1]` fd array + `NAME_PID` + `$!` + a job, auto-unset on exit.

**Architecture:** A new `Command::Coproc { name, body }` AST node. The executor spawns the body via the existing `fork_and_run_in_subshell` with the two pipe ends as the child's stdin/stdout, holds the other ends (relocated to high fds, close-on-exec) in a NAME-keyed `Shell.coprocs: Vec<Coproc>`, and publishes `NAME`/`NAME_PID`/`$!`/a job. The reap path (`reap_completed`) auto-unsets a coproc when its pid dies. Single active coproc (warn on a second); state is multi-ready.

**Tech Stack:** Rust, `libc` (`pipe`, `dup2`, `close`, `fcntl`, reused v156 `alloc_high_fd`/`make_pipe`/`set_cloexec`).

**Spec:** `docs/superpowers/specs/2026-06-14-coproc-design.md`. Read it first.

```rust
/// Parses a `coproc` body after the `coproc` keyword has been consumed.
/// `coproc NAME compound` → named; otherwise anonymous (`COPROC`), body parsed
/// as an ordinary command (simple or compound).
fn parse_coproc_command(iter: &mut TokenCursor) -> Result<Command, ParseError> {
    // Named form: a plain Word followed by a token that starts a compound command.
    if let Some(Token::Word(w)) = iter.peek() {
        if let Some(nm) = word_literal_text(w) {
            // is the NEXT-next token a compound-command opener?
            if is_compound_opener(iter.peek2()) {
                let nm = nm.to_string();
                iter.next(); // consume the NAME word
                let body = parse_command_inner(iter)?;
                return Ok(Command::Coproc { name: nm, body: Box::new(body) });
            }
        }
    }
    // Anonymous: parse the body (simple or compound) as an ordinary command.
    let body = parse_command_inner(iter)?;
    Ok(Command::Coproc { name: "COPROC".to_string(), body: Box::new(body) })
}

/// True if `tok` is the first token of a compound command
/// (`{`, `(`, if/while/until/for/case/select, `[[`, `((`).
fn is_compound_opener(tok: Option<&Token>) -> bool {
    match tok {
        Some(Token::Op(Operator::LParen)) => true,           // ( subshell )
        Some(Token::ArithBlock(_)) => true,                  // (( ))
        Some(t) => matches!(
            keyword_of(t),
            Some(Keyword::LBrace) | Some(Keyword::If) | Some(Keyword::While)
            | Some(Keyword::Until) | Some(Keyword::For) | Some(Keyword::Case)
            | Some(Keyword::Select) | Some(Keyword::DoubleBracketOpen)
        ),
        None => false,
    }
}
```
NOTE: `parse_command_inner` for the body skips its own leading newlines and handles `[[`/`((`/`{`/etc. + simple commands. If `TokenCursor` lacks a `peek2()` (two-token lookahead), add one (mirror `peek`/`current_line`; it returns the token after the next without consuming). Check `src/command.rs` for an existing 2-token lookahead first and reuse it.

```rust
impl Shell {
    /// Called from the reap path when child `pid` has exited: if it is a live
    /// coproc, close its held fds, unset NAME + NAME_PID, and drop the record.
    /// bash unsets the coproc variables once the coprocess is reaped.
    pub fn reap_coproc(&mut self, pid: libc::pid_t) {
        let Some(idx) = self.coprocs.iter().position(|c| c.pid == pid) else { return; };
        let c = self.coprocs.remove(idx);
        unsafe {
            libc::close(c.read_fd);
            libc::close(c.write_fd);
        }
        self.unset(&c.name);                       // the NAME array
        self.unset(&format!("{}_PID", c.name));    // NAME_PID
    }
}
```

```rust
/// Start a coprocess: fork the body with its stdin/stdout wired to two pipes,
/// hold the shell-side ends (relocated to high fds, close-on-exec) as NAME[0]
/// (read) / NAME[1] (write), publish NAME_PID, $!, and a job. Returns 0 on a
/// successful spawn (the coproc runs asynchronously), 1 on a pipe/fork failure.
fn run_coproc(name: &str, body: &Command, shell: &mut Shell) -> ExecOutcome {
    use std::os::unix::io::RawFd;
    // bash warns (but proceeds) if a coproc is already live (v157 single-active).
    if !shell.coprocs.is_empty() {
        let existing = &shell.coprocs[0];
        eprintln!(
            "huck: warning: execute_coproc: coproc [{}:{}] still exists",
            existing.pid, existing.name
        );
    }
    // pipe_in: shell writes in_w -> coproc reads in_r (its stdin).
    // pipe_out: coproc writes out_w (its stdout) -> shell reads out_r.
    let (in_r, in_w) = match make_pipe() { Ok(p) => p, Err(e) => { eprintln!("huck: coproc: {e}"); return ExecOutcome::Continue(1); } };
    let (out_r, out_w) = match make_pipe() {
        Ok(p) => p,
        Err(e) => { unsafe { libc::close(in_r); libc::close(in_w); } eprintln!("huck: coproc: {e}"); return ExecOutcome::Continue(1); }
    };
    flush_stdout();
    // Fork the body: child stdin = in_r, stdout = out_w, stderr inherited; own
    // process group (pgid_target 0); the child must NOT keep the shell's ends.
    let pid = match fork_and_run_in_subshell(
        body, shell, in_r, out_w, libc::STDERR_FILENO, 0, &[in_w, out_r], None, None,
    ) {
        Ok(pid) => pid,
        Err(e) => { unsafe { libc::close(in_r); libc::close(in_w); libc::close(out_r); libc::close(out_w); } eprintln!("huck: coproc: {e}"); return ExecOutcome::Continue(1); }
    };
    // Parent: close the child ends; relocate the shell ends high + cloexec.
    unsafe { libc::close(in_r); libc::close(out_w); }
    let relocate = |fd: RawFd| -> io::Result<RawFd> {
        let hi = alloc_high_fd(fd)?;   // F_DUPFD from 10
        unsafe { libc::close(fd); }
        set_cloexec(hi);               // coproc fds don't leak into unrelated children
        Ok(hi)
    };
    let read_fd = match relocate(out_r) {
        Ok(f) => f,
        Err(e) => { unsafe { libc::close(out_r); libc::close(in_w); } eprintln!("huck: coproc: {e}"); return ExecOutcome::Continue(1); }
    };
    let write_fd = match relocate(in_w) {
        Ok(f) => f,
        // read_fd is ALREADY relocated (must close the relocated fd); in_w not yet.
        Err(e) => { unsafe { libc::close(read_fd); libc::close(in_w); } eprintln!("huck: coproc: {e}"); return ExecOutcome::Continue(1); }
    };
    // Publish: NAME=(read write), NAME_PID, $!, a job, and the record.
    let mut elems = std::collections::BTreeMap::new();
    elems.insert(0usize, read_fd.to_string());
    elems.insert(1usize, write_fd.to_string());
    let _ = shell.replace_array(name, elems);
    shell.set(&format!("{name}_PID"), pid.to_string());
    shell.last_bg_pid = Some(pid);
    shell.jobs.add(pid, vec![pid], format!("coproc {name}"));
    shell.coprocs.push(crate::shell_state::Coproc { name: name.to_string(), pid, read_fd, write_fd });
    ExecOutcome::Continue(0)
}
```
Confirm `make_pipe`, `alloc_high_fd`, `set_cloexec`, `flush_stdout`, `fork_and_run_in_subshell` signatures (all exist from v156/earlier). `replace_array` takes `(name, BTreeMap<usize, String>)` — verify the exact key type (usize vs u64) and adjust. `shell.jobs.add(pid, pids, display)` and `shell.last_bg_pid` exist.
