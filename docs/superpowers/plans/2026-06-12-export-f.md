```rust
/// Securely parse a `BASH_FUNC_<name>%%` env value into a function body.
/// Reconstructs `"{name} {value}"` (= `name () { body }`) and parses it,
/// accepting ONLY a single `FunctionDef` whose name matches, with nothing after
/// the `}` and no background. NEVER executes the value. Returns `None` to skip
/// (parse error, trailing tokens, non-function, invalid name) — Shellshock-safe.
fn parse_imported_function(name: &str, value: &str) -> Option<Box<crate::command::Command>> {
    // name must be a plain identifier (huck functions are POSIX identifiers).
    if name.is_empty()
        || !name.chars().next().map(|c| c == '_' || c.is_ascii_alphabetic()).unwrap_or(false)
        || !name.chars().all(|c| c == '_' || c.is_ascii_alphanumeric())
    {
        return None;
    }
    let src = format!("{name} {value}");
    let tokens = crate::lexer::tokenize(&src).ok()?;
    let seq = crate::command::parse(tokens).ok()??;
    if !seq.rest.is_empty() || seq.background {
        return None;
    }
    match seq.first {
        crate::command::Command::FunctionDef { name: n, body } if n == name => Some(body),
        _ => None,
    }
}
```

## Notes for the implementer
- **Security is the load-bearing part:** `parse_imported_function` must NEVER execute the env value and must reject anything but a single `FunctionDef` (trailing tokens, background, name mismatch, parse error all → `None`). The `shellshock_*` tests are mandatory.
- **`BASH_FUNC_*%%` env vars are NOT ordinary variables** — skipped on load, re-emitted at spawn from the live exported set. A normal `FOO=bar` must still load as a variable (regression-check `Shell::new`).
- **`exported_function_env` is computed at spawn time** from the current set — a function exported then re-defined emits the new body; an `unset -f` one is dropped.
- **Don't change `declare -f`/`declare -F`** — only the `-f`+`-x` combination is new.
- The BODY value is v146-normalized (re-parseable, not bash-byte-identical) — the harness compares only the env-KEY shape + `declare -fx` trailer, never the body bytes.
